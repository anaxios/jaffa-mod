Installation

To set up and play our modded Minecraft server, follow these detailed steps:

Step 1: Download and Set Up Prism Launcher

  1. Download Prism Launcher - Go to https://prismlauncher.org/ and download the latest version of Prism Launcher for your operating system.

  2. Log In to Your Minecraft Account - Open Prism Launcher. - Log in with your Minecraft account credentials.

  3. Create a New Instance - In Prism Launcher, click on the “Instances” tab. - Select “Add Instance” to create a new instance.

  4. Choose Modrinth Modpacks - In the instance creation window, select “Modrinth” as the source for modpacks. - Browse or search for modpacks.

  5. Search for ilguappo’s Cave Game - In the search bar, type “ilguappo’s Cave Game” to find the specific modpack. - Select and install the modpack.

Step 2: Install Node.js

  1. Download Node.js - Visit https://nodejs.org/en/download/package-manager. - Follow the instructions to install Node.js for your operating system.

  2. Install Holesail - Open a terminal. - Run the following command to install Holesail globally: - For Windows: bash npm i holesail -g - For Mac or Linux: sh sudo npm i holesail -g

  3. Run Holesail - In the terminal, paste the following command and press Enter: bash holesail --port 25565 revisable-easeful-hurried-doorknob-golf-handwork-sureness-tropics-stinking-grape-mousy-lustfully-glimpse-overcast-counting-litigate

Step 3: Start Minecraft Instance

  1. Launch the Minecraft Instance - Open Prism Launcher. - Select the instance you created with the ilguappo’s Cave Game modpack. - Click “Play” to start the game.

  2. Add New Server in Minecraft - In the Minecraft main menu, click “Multiplayer.” - Click “Add Server.” - For the server address, enter: 127.0.0.1. - Save and connect to your local server.

You are now set up to play on our modded Minecraft server using the ilguappo’s Cave Game modpack!
